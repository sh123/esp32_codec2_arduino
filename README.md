# ESP32 Codec2 Arduino Library
This is latest non-development version of Codec2 library packaged for ESP32 Arduino usage.

It is based on latest stable version from https://github.com/drowe67/codec2-dev repository.

Version in https://github.com/drowe67/codec2 considered development version and may stop supporting some of low bit rate modes in favor of RADEV1 codec according to author.
